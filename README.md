# Zomato Data Analysis

## 📊 Project Overview
This project performs exploratory data analysis on Zomato dataset to extract insights about restaurants, cuisines, and customer preferences.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Insights
- Online delivery increases restaurant popularity
- Mid-range restaurants get better ratings
- North Indian cuisine is most popular

## 📂 Files
- zomato_analysis.ipynb → Analysis notebook
- presentation.pptx → Project presentation
- dataset.csv → Dataset used
